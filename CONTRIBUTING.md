# Contributing to BroShow

Thanks for your interest in contributing! All pull requests are welcome.

## How to Contribute

1. Fork the repository.
2. Create a feature branch (`git checkout -b my-feature`).
3. Make your changes.
4. Commit your changes (`git commit -am 'Add some feature'`).
5. Push to the branch (`git push origin my-feature`).
6. Open a pull request against `main`.

## Development Setup

```bash
npm install
npm run build
```

Load the unpacked extension from the `dist/` directory in your browser's
extension management page.

## Running Tests

```bash
npm test
```

## Reporting Bugs

Open an issue on [GitHub](https://github.com/jeffabailey/broshow/issues) with:

- A clear description of the bug.
- Steps to reproduce.
- Expected vs. actual behavior.
- Browser version and OS.

## Suggesting Features

Open an issue describing the feature, why it's useful, and how it should
behave.
